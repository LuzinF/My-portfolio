1) Посмотреть где находишься - pwd
2) Создать папку - mkdir
3) Зайти в папку - cd namefolder
4) Создать 3 папки - mkdir name1 name2 name 3
5) Зайти в любую папку - cd path
6) Создать 5 файлов - touch file1 file2 file3 file4 file5
7) Создать 3 папки - mkdir folder1 folder2 folder3
8) Вывести список содержимого папки - ls -l
9) Открыть любой txt файл - vi filename
10) Написать туда что-нибудь - клавишей i зайти в режим редактирования, Esc выйти из этого режима
11) Сохранить и выйти - прописать команду :wq сохранить и выйти
12) Выйти из папки на уровень выше - cd ..
13) Переместить любые 2 файла - команда mv. Для нескольких файлов нужно встать в папку с этими файлами и написать file1 file2 path/to/the/directory
14) Копировать любые 2 файла - команда cp, работает по схожей схеме как mv
15) Найти файл по имени - find path -name file или если находимся в нужной папке то вместо path можно написать ./
16) Просмотреть содержимое в реальном времени - grep "word" file
17) Вывести несколько первых строк из текстового файла - head -n file (через тире указываем кол-во строк)
18) Вывести несколько последних строк - tail -n file (через тире указываем кол-во строк)
19) Просмотреть содержимое длинного файла - команда просмотра файла less, открывается доп. окно с файлом где можно листать файл, писать команды (поиск, удаление, настройки просмотра)
20) Вывести дату и время - date +%D-%T (опции чтобы отображалось только дата и время)
21) Команда open - команда start name, она открывает файл/папку
22*) Отправить http запрос на сервер - 
curl "http://162.55.220.72:5005/object_info_3?name=Vadim&age=32&salary=1000"
curl -G -d "name=Vadim&age=32&salary=1000" http://162.55.220.72:5005/object_info_3
